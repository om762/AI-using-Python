It it simple project for solving Maze Problem.



## Authors

## Run Locally


Clone the project

```bash
  git clone https://github.com/om762/AI-using-Python
```

Go to the project directory

```cmd
  cd AI-using-Python
```

Install dependencies

```cmd
  pip install Pillow
```

Run the project using

```cmd
  py maze.py maze.txt
```
